# 2021 컴퓨터네트워크 설계과제

### Getting started
가상환경 설정:

```bash
python -m venv venv
``` 

가상환경 실행:

```bash
(Linux) source ./venv/bin/activate
(Window) ./venv/Scripts/Activate
```

필요한 패키지 설치:

```bash
python -m pip install -r requirements.txt
```

sniffer.py 실행:

```bash
python sniffer.py
```
1번을 누르면 packet sniffing을 시작합니다.  
2번을 누르면 packet sniffing을 종료합니다. 그리고 결과값을 알려줍니다.  

```output```폴더에 결과가 저장됩니다.
